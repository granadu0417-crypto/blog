## ✍️ 콘텐츠 생성 워크플로우

### 방법 1: 자동 생성 스크립트 (추천)

가장 빠르게 시작하는 방법입니다:

```bash
npm run generate-post
```

이 명령어는:
- ✅ 카테고리별로 샘플 게시글 생성
- ✅ Unsplash/Pexels에서 자동으로 이미지 검색
- ✅ SEO에 최적화된 메타데이터 자동 추가
- ✅ 읽기 시간 자동 계산

### 방법 2: 수동으로 게시글 작성

1. **카테고리 폴더 확인**

```
posts/
├── finance/        # 금융/투자
├── tech/          # IT/테크
├── health/        # 건강/웰빙
├── realestate/    # 부동산/인테리어
├── education/     # 교육/자기계발
├── lifestyle/     # 생활정보
├── travel/        # 여행/맛집
├── hobby/         # 취미/라이프스타일
├── shopping/      # 쇼핑/리뷰
└── trend/         # 트렌드/뉴스
```

2. **게시글 템플릿**

`posts/[category]/your-post-title.md` 파일 생성:

```markdown
---
title: "게시글 제목 (50-60자, 키워드 포함)"
date: "2025-01-15"
excerpt: "게시글 요약 (150-160자, 검색 결과에 표시됨)"
category: "tech"
tags: ["프로그래밍", "AI", "ChatGPT"]
imageUrl: "https://images.unsplash.com/..."
author: "블로그 관리자"
---
```

## 🔍 SEO 최적화 체크리스트

### 필수 체크리스트

- [ ] **제목 최적화** (50-60자)
  - [ ] 숫자 포함 (예: "2025년 TOP 10")
  - [ ] 주요 키워드 앞쪽 배치
  - [ ] 감정/혜택 표현 (예: "완벽", "쉬운", "무료")

- [ ] **메타 설명** (150-160자)
  - [ ] 핵심 키워드 2-3개 포함
  - [ ] CTA 포함 (예: "지금 확인하세요")
  - [ ] 독자 혜택 명확히

- [ ] **본문 구조**
  - [ ] H2, H3 태그로 구조화
  - [ ] 최소 2,000-3,000자
  - [ ] 단락당 3-4문장
  - [ ] 리스트/표 활용

- [ ] **이미지 최적화**
  - [ ] Alt 텍스트 작성 (키워드 포함)
  - [ ] 고품질 이미지 (최소 1200px 폭)
  - [ ] WebP 또는 JPEG 형식
  - [ ] 크레딧 표시

- [ ] **내부 링크**
  - [ ] 관련 게시글 3-5개 링크
  - [ ] 카테고리 페이지 링크
  - [ ] 앵커 텍스트에 키워드 포함

- [ ] **외부 링크**
  - [ ] 권위 있는 사이트 1-2개 링크
  - [ ] rel="noopener" 속성 추가

## 💰 구글 애드센스 준비

### 승인 조건

1. **콘텐츠 요구사항**
   - ✅ 최소 20-30개의 고품질 게시글
   - ✅ 각 게시글 2,000자 이상
   - ✅ 독창적이고 유용한 콘텐츠
   - ✅ 정기적인 업데이트 (주 3-5개)

2. **필수 페이지** (이미 준비됨 ✅)
   - ✅ 소개 페이지 (`/about`)
   - ✅ 연락처 페이지 (`/contact`)
   - ✅ 개인정보처리방침 (`/privacy`)
   - ✅ 이용약관 (`/terms`)

3. **기술적 요구사항**
   - ✅ 반응형 디자인
   - ✅ 빠른 로딩 속도 (<3초)
   - ✅ HTTPS 적용
   - ✅ 모바일 친화적

### 신청 프로세스

1. **Google AdSense 가입**
   - https://www.google.com/adsense/ 접속
   - 블로그 URL 입력: `https://blog-7xx.pages.dev`
   - 정보 입력 및 약관 동의

2. **사이트 연결**
   - AdSense 코드를 `app/layout.tsx`에 추가:
   ```tsx
   <script 
     async 
     src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXX"
     crossOrigin="anonymous"
   />
   ```

3. **승인 대기** (보통 1-2주)
   - 이 기간 동안 계속 게시글 발행
   - 품질 유지 및 트래픽 증대 노력

## 📈 수익 극대화 전략

### 1. 콘텐츠 발행 주기

```
초기 3개월 (승인 목표):
- 주 5-7개 게시글
- 카테고리별 고르게 분산
- 롱테일 키워드 집중 공략

이후 (안정적 수익):
- 주 3-5개 신규 게시글
- 상위 20개 글 월 1회 업데이트
- 계절/트렌드에 맞춘 콘텐츠
```

### 2. 고수익 카테고리 집중

**CPC가 높은 카테고리** (우선순위):
1. 💰 금융/투자 ($2-$5)
2. 💻 IT/테크 ($1.5-$3)
3. 🏠 부동산/인테리어 ($1-$2.5)
4. 📚 교육/자기계발 ($1-$2)
5. 🏃 건강/웰빙 ($0.8-$1.5)

**트래픽 확보 카테고리** (보조):
6. ✨ 생활정보 ($0.5-$1)
7. ✈️ 여행/맛집 ($0.3-$0.8)
8. 🛍️ 쇼핑/리뷰 ($0.5-$1.2)
9. 🎨 취미/라이프스타일 ($0.3-$0.7)
10. 🔥 트렌드/뉴스 ($0.2-$0.6)

### 3. 키워드 전략

**롱테일 키워드 공략**:
```
❌ 나쁜 예: "주식"
✅ 좋은 예: "2025년 초보자를 위한 주식 투자 시작 방법"

❌ 나쁜 예: "다이어트"
✅ 좋은 예: "직장인을 위한 점심 도시락 다이어트 식단"
```

**키워드 연구 도구**:
- Google Keyword Planner
- Google Trends
- Naver 키워드 도구
- 네이버/구글 자동완성 키워드

### 4. 광고 배치 최적화

승인 후 권장 광고 위치:

```
┌─────────────────────┐
│   헤더/네비게이션    │
├─────────────────────┤
│   첫 문단 (100자)    │
├─────────────────────┤
│   [자동 광고 1]     │  ← 클릭률 높음
├─────────────────────┤
│   본문 내용         │
│   (1000자)          │
├─────────────────────┤
│   [디스플레이 광고] │  ← 필수 배치
├─────────────────────┤
│   본문 계속         │
│   (1000자)          │
├─────────────────────┤
│   [관련 콘텐츠]     │  ← 추가 클릭 유도
├─────────────────────┤
│   댓글/푸터         │
└─────────────────────┘
```

### 5. 트래픽 증대 방법

**Google Search Console**:
1. 사이트 등록: https://search.google.com/search-console
2. 소유권 확인
3. sitemap.xml 제출: `https://blog-7xx.pages.dev/sitemap.xml`
4. 색인 상태 모니터링

**네이버 서치어드바이저**:
1. 사이트 등록: https://searchadvisor.naver.com
2. 소유권 확인
3. 사이트맵 제출

**소셜 미디어**:
- 페이스북 페이지 운영
- 인스타그램 연동
- 네이버 블로그/카페 홍보
- 카카오톡 오픈채팅

### 6. 수익 모니터링

**주요 지표**:
- **페이지 뷰** (PV): 목표 1,000/일
- **클릭률** (CTR): 목표 1-3%
- **클릭당 수익** (CPC): 카테고리별 상이
- **RPM** (1000뷰당 수익): 목표 $1-$5

**월 수익 목표 예시**:
```
트래픽: 30,000 PV/월
CTR: 2%
CPC: $0.50
───────────────────
월 수익: 30,000 × 0.02 × $0.50 = $300
년 수익: $3,600
```
