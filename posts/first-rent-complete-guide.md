# 첫 전세/월세 완벽 가이드

"처음 집을 구하는데 뭐부터 해야 할지 모르겠어요..." 걱정 마세요. 이 글 하나면 충분합니다. 매물 찾기부터 계약, 입주까지 모든 과정을 단계별로 알려드립니다.

## 3분 요약

- **예산 설정**: 월급의 30% 이내 (보증금 포함)
- **매물 찾기**: 직방, 다방, 네이버 부동산 + 발품
- **필수 확인**: 등기부등본, 건축물대장, 실거주 확인
- **계약 시기**: 월말 or 이사철 피하기 (협상 유리)
- **사기 예방**: 집주인 신분증, 임대차 3법 확인

## 전세 vs 월세 선택하기

### 전세와 월세 차이

**전세**
- **개념**: 보증금만 내고 월세 없음
- **보증금**: 1-5억원
- **장점**: 월 고정비 없음, 목돈 불리기
- **단점**: 초기 자금 부담, 계약 만기 시 이사

**월세**
- **개념**: 보증금 + 월세
- **보증금**: 500-3,000만원
- **월세**: 30-100만원
- **장점**: 초기 자금 부담 적음
- **단점**: 월 고정비 부담

### 내게 맞는 선택은?

**전세 추천**
- ✅ 목돈 있음 (부모님 지원)
- ✅ 2년 이상 거주 예정
- ✅ 월 고정비 부담 싫음
- ✅ 안정적인 직장

**월세 추천**
- ✅ 목돈 없음 (사회 초년생)
- ✅ 단기 거주 (1-2년)
- ✅ 유동성 필요
- ✅ 직장 이동 가능성

**보증금 전환 계산**
```
보증금 1,000만원 증가 = 월세 5만원 감소
(금리 6% 기준)

예시:
A. 보증금 3,000만원 / 월세 50만원
B. 보증금 5,000만원 / 월세 40만원

2,000만원 증가 → 10만원 감소
```

## 예산 설정하기

### 적정 주거비

**월급 대비 비율**
```
월급: 300만원

30% 룰:
전세: 보증금 1억 (이자 50만원)
월세: 보증금 3,000만원 + 월세 60만원
```

**숨은 비용 포함**
```
월세: 50만원
+ 관리비: 15만원
+ 전기/가스/수도: 5만원
+ 인터넷/TV: 5만원
───────────────────
실제 비용: 75만원
```

### 초기 비용 준비

**전세 계약 시**
```
보증금: 10,000만원
+ 중개 수수료: 40만원 (0.4%)
+ 이사 비용: 50만원
+ 등기비용: 30만원
───────────────────
총: 10,120만원
```

**월세 계약 시**
```
보증금: 3,000만원
+ 중개 수수료: 30만원
+ 이사 비용: 50만원
+ 첫 달 월세: 50만원
───────────────────
총: 3,130만원
```

## 매물 찾기 전략

### 온라인 플랫폼

**직방 (Zigbang)**
- **장점**: 사진 많음, 필터 상세
- **추천**: 원룸, 오피스텔
- **팁**: "실사진" 필터 활용

**다방**
- **장점**: 매물 많음
- **추천**: 빌라, 아파트
- **팁**: "확인매물" 표시 확인

**네이버 부동산**
- **장점**: 실거래가 확인
- **추천**: 아파트
- **팁**: "단지 정보" 상세 확인

**호갱노노 (앱)**
- **장점**: 실거래가 비교
- **추천**: 시세 파악
- **팁**: 주변 시세 확인 필수

### 오프라인 발품

**부동산 중개소 방문**
- **장점**: 온라인 없는 매물
- **단점**: 시간 소요

**방문 팁**
```
1. 동네 큰 중개소 3-4곳
2. "이 조건으로 매물 있나요?" 직접 물어보기
3. 명함 받고 연락처 저장
4. "나중에 연락 주세요" 부탁
```

**동네 탐색**
- 직접 걸어다니며 "임대" 현수막 확인
- 이른 아침 or 주말 탐색

### 매물 검색 꿀팁

**검색 조건 설정**
```
필수:
□ 지역 (역세권 10분 이내)
□ 가격 (예산 ±10%)
□ 방 개수 (원룸/투룸)
□ 층수 (2층 이상)

선택:
□ 엘리베이터
□ 주차 가능
□ 반려동물
□ 풀옵션
```

**숨은 비용 확인**
- 관리비 (동 여름 겨울 다름)
- 주차비 (월 5-10만원)
- 인터넷 (자체 설치 or 포함)

## 계약 전 필수 확인

### 등기부등본 확인 (필수!)

**발급 방법**
- 인터넷 등기소 (www.iros.go.kr)
- 수수료: 700원
- 즉시 발급

**확인 사항**
```
갑구 (소유권):
□ 현재 소유자 = 계약할 집주인
□ 가압류, 압류 여부 (없어야 함)
□ 근저당 금액 (보증금보다 적어야 함)

을구 (권리 관계):
□ 전세권 설정 여부
□ 기존 임차인 여부
```

**위험 신호**
- ❌ 근저당 > 보증금 (집주인 빚 많음)
- ❌ 가압류/압류 (경매 가능성)
- ❌ 소유자 ≠ 계약자 (사기 가능성)

### 건축물대장 확인

**확인 사항**
- 용도 (주택, 상가주택)
- 면적 (실제 면적 확인)
- 사용 승인일 (건물 연식)
- 불법 증축 여부

### 임대차 3법 확인

**전월세 상한제**
- 재계약 시 5% 이상 인상 불가

**계약 갱신 요구권**
- 1회 연장 가능 (총 4년 거주)

**전월세 신고제**
- 보증금 6,000만원 or 월세 30만원 이상
- 계약 후 30일 이내 신고

## 계약서 작성 주의사항

### 계약 시 준비물

**임차인 (나)**
- 신분증
- 인감도장 (없으면 서명)
- 보증금 (계좌이체 준비)

**임대인 (집주인)**
- 신분증 (사본)
- 인감증명서
- 등기권리증
- 통장 (보증금 입금용)

### 표준 임대차 계약서

**필수 기재 사항**
```
□ 임대인/임차인 인적사항
□ 부동산 소재지 (정확한 주소)
□ 임대 면적 (㎡)
□ 보증금/월세 금액
□ 계약 기간 (보통 2년)
□ 특약 사항
□ 계약일, 잔금일, 입주일
□ 중개 수수료
```

### 특약 사항 꼭 쓰기

**추천 특약**
```
1. "임대인은 임차인 입주 전까지 도배, 장판,
   화장실 청소를 완료한다"

2. "에어컨, 냉장고 등 옵션 제품 고장 시
   임대인이 수리 또는 교체한다"

3. "관리비에 포함되는 항목:
   공용 관리비, 수도세, 인터넷"

4. "임대인은 임의로 방문할 수 없으며,
   방문 시 3일 전 사전 통지한다"

5. "임차인은 계약 만료 1개월 전 통보 시
   중도 해지 가능하다" (협상 필요)
```

### 계약금/잔금 지급

**지급 일정**
```
계약금 (10%):
계약 당일 지급

잔금 (90%):
입주일에 지급 (계약 후 2-4주)
```

**안전한 지급 방법**
- 계좌이체 (현금 X)
- 집주인 명의 통장 확인
- 입금증 보관
- 영수증 받기

**확정일자 받기 (필수!)**
- 계약 당일 or 입주 당일
- 관할 주민센터 or 등기소
- 수수료: 600원
- 효력: 경매 시 우선 변제

- [ ] 화장실 청소

- [ ] 창문 청소

**옵션 작동 확인**
- [ ] 에어컨 냉난방
- [ ] 냉장고 작동
- [ ] 세탁기 작동
- [ ] 가스레인지
- [ ] 온수 (보일러)
- [ ] 조명 전체

**상태 사진 촬영**
- 입주 전 상태 증거 (퇴실 시 분쟁 방지)
- 기존 흠집, 얼룩 사진

### 입주 시 신고

**주민등록 전입신고**
- 입주 후 14일 이내
- 관할 주민센터
- 필요 서류: 신분증, 계약서

**전입신고 전 확인**
- 확정일자 먼저 받기
- 전입신고 → 확정일자 (순서 중요!)

### 공과금 처리

**명의 변경**
```
전기: 한국전력공사 (국번 없이 123)
가스: 지역 도시가스 (고객센터)
수도: 지역 수도사업소
인터넷/TV: 통신사 (개별 계약)
```

**팁**
- 입주 3일 전 신청
- 기존 명의 잔액 확인 (정산)

## 사기 피하는 법

### 흔한 사기 유형

**1. 전세 사기**
- 집주인이 근저당 많이 설정
- 경매로 집 날아감
- 보증금 못 받음

**예방법**
- 등기부등본 필수 확인
- 근저당 < 보증금
- 확정일자 + 전입신고

**2. 이중 계약**
- 같은 집에 2명과 계약
- 먼저 계약한 사람만 인정

**예방법**
- 계약 즉시 확정일자
- 전입신고 빠르게

**3. 가짜 집주인**
- 실제 집주인 행세
- 계약금만 받고 잠적

**예방법**
- 신분증 + 등기부등본 대조
- 등기권리증 확인
- 부동산 통해 계약

### 주의해야 할 매물

**위험 신호**
```
❌ 시세보다 너무 저렴 (20% 이상)
❌ "급해요", "빨리 계약해요" 압박
❌ 집주인 직접 계약 고집
❌ 현금 거래만 요구
❌ 집주인 신분 확인 거부
❌ 계약서 없이 계약금 먼저 요구
```

## 분쟁 발생 시 대처

### 보증금 안 돌려줄 때

**절차**
```
1. 내용증명 발송 (우체국)
2. 임대차 분쟁 조정 신청 (주택임대차분쟁조정위원회)
3. 소액사건심판 (3,000만원 이하)
4. 민사소송 (변호사)
```

**예방**
- 계약 만료 3개월 전 통보
- 문자/카톡으로 증거 남기기

### 하자 수리 거부

**대처법**
- 특약에 명시된 경우: 내용증명
- 명시 안 된 경우: 협의
- 긴급 상황: 본인 수리 후 청구

## 자주 묻는 질문

### Q1. 월세를 전세로 전환 가능한가요?
A. 가능합니다. 집주인과 협의하세요. 보증금 1,000만원 = 월세 약 5만원 (금리 6% 기준)입니다. 전환 시 새 계약서 작성하고 확정일자 다시 받으세요.

### Q2. 반려동물 키워도 되나요?
A. 집주인 동의 필요합니다. 계약 전 꼭 말하고, 특약에 "반려동물 사육 가능"이라고 명시하세요. 나중에 문제 되면 퇴거 요구받을 수 있습니다.

### Q3. 중개 수수료는 얼마인가요?
A. 전세/월세 모두 0.4-0.5%입니다. 5,000만원이면 20-25만원입니다. 세입자와 집주인이 각각 부담합니다. 영수증 꼭 받으세요.

### Q4. 전입신고 안 하면 어떻게 되나요?
A. 대항력이 없어 경매 시 보증금 못 받습니다. 확정일자 효력도 없습니다. 입주 후 14일 이내 필수로 하세요. 늦으면 과태료 5만원입니다.

### Q5. 계약 기간 중 이사 가고 싶으면?
A. 원칙적으로 불가능합니다. 하지만 협의 가능합니다. 특약에 "중도 해지 가능" 조건을 넣거나, 대신 살 사람을 구해서 집주인 동의받으세요.

## 마무리

첫 집 구하기는 어렵지만, 제대로 알고 하면 안전합니다. 서두르지 말고, 꼼꼼히 확인하세요.

**계약 전 3대 원칙**
1. **등기부등본**: 근저당 < 보증금
2. **확정일자**: 계약 당일 or 입주 당일
3. **전입신고**: 입주 후 14일 이내

"Measure twice, cut once."
