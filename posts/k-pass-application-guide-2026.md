## K패스란?

### 기본 개념

K패스는 대중교통을 **월 15회 이상** 이용하면 교통비 일부를 **환급**해주는 정부 지원 제도입니다.

| 구분 | 내용 |
|------|------|
| **시행일** | 2024년 5월 1일~ |
| **운영 기관** | 국토교통부 |
| **핵심 혜택** | 대중교통비 환급 |
| **환급률** | 최대 53% |

### 알뜰교통카드와 차이점

| 항목 | 알뜰교통카드 | K패스 |
|------|-------------|-------|
| **환급 방식** | 걷기/자전거 마일리지 | 이용 횟수 기반 |
| **이용 조건** | 이동수단 등록 필요 | 15회 이상 이용 |
| **환급률** | 최대 30% | **최대 53%** |
| **복잡도** | 앱 사용 필수 | 카드만 사용 |

> **핵심**: K패스는 걷기 안 해도 대중교통만 타면 환급됩니다!

## K패스 신청 방법

### 1단계: 카드 발급

**K패스 제휴 카드를 발급**받아야 합니다.

**제휴 카드사:**

| 카드사 | 카드명 | 특징 |
|--------|--------|------|
| **신한카드** | K패스 신한카드 | 대중교통 추가 적립 |
| **KB국민** | K패스 국민카드 | 포인트 통합 |
| **우리카드** | K패스 우리카드 | 캐시백 혜택 |
| **하나카드** | K패스 하나카드 | 마일리지 적립 |
| **NH농협** | K패스 농협카드 | 주유 할인 |
| **BC카드** | K패스 BC카드 | 다양한 제휴 |

**발급 방법:**
1. 각 카드사 앱 또는 홈페이지 접속
2. 'K패스' 검색
3. 카드 신청 및 본인인증
4. 카드 수령 (약 5~7일)

### 2단계: K패스 앱 가입

**K패스 앱 설치 및 가입**

| 단계 | 방법 |
|------|------|
| 1 | 앱스토어에서 'K패스' 검색 |
| 2 | 앱 설치 |
| 3 | 회원가입 (본인인증) |
| 4 | K패스 카드 등록 |

### 3단계: 카드 등록

1. K패스 앱 로그인
2. '카드 등록' 메뉴 선택
3. 발급받은 K패스 카드 정보 입력
4. 등록 완료

## 자주 묻는 질문

**Q. 기존 교통카드로 사용 가능한가요?**

A. **불가능합니다.** K패스 제휴 카드를 새로 발급받아야 합니다. 기존 신용/체크카드는 K패스 혜택을 받을 수 없습니다.

**Q. 월 15회 미만 이용하면?**

A. **환급이 없습니다.** 15회 이상 이용해야 환급 대상이 됩니다. 14회 이하는 일반 교통카드와 동일합니다.

**Q. 택시도 환급되나요?**

A. **택시는 제외**됩니다. 버스, 지하철, GTX, 광역버스 등 대중교통만 해당됩니다.

**Q. 카드 여러 장 등록 가능한가요?**

A. **1인 1카드**만 등록 가능합니다. 여러 카드를 번갈아 사용하면 환급 조건(15회)을 충족하기 어려울 수 있습니다.

**Q. 청년 혜택은 언제까지?**

A. **만 19세~34세**까지입니다. 35세가 되는 해부터는 일반 환급률(20%)이 적용됩니다.

**Q. 환급은 어떻게 받나요?**

A. 카드사에 따라 **청구 할인** 또는 **포인트 적립**으로 지급됩니다. 카드 명세서에서 확인할 수 있습니다.
