# 2026년 건강검진 대상자 조회, 무료 국가검진 항목과 예약 방법

─────────────────────────────────────

## 💡 3줄 요약

1. **2026년 대상자**: 짝수 해 출생자 (1960, 1970, 1980, 1990년생 등)
2. **무료 검진**: 일반건강검진 + 5대 암검진 (국가 지원)
3. **조회 방법**: 국민건강보험공단 홈페이지 또는 The건강보험 앱

## 📊 2026년 건강검진 대상자

### 일반건강검진 대상

| 구분 | 대상자 |
|------|--------|
| **직장가입자** | 비사무직: 매년 / 사무직: 짝수년 출생자 |
| **지역가입자** | 세대주 및 만 40세 이상 세대원 (짝수년 출생) |
| **피부양자** | 만 40세 이상 (짝수년 출생) |
| **의료급여수급권자** | 만 19~64세 (짝수년 출생) |

### 쉽게 확인하기

```
🗓️ 2026년 대상 출생 연도 (짝수년)
→ 1946, 1956, 1966, 1976, 1986, 1996, 2006년생 등

❌ 2026년 비대상 (홀수년)
→ 1947, 1957, 1967, 1977, 1987, 1997, 2007년생 등
```

> ⚠️ 비사무직 근로자는 **매년** 검진 대상!

## 🔍 대상자 조회 방법

### 방법 1️⃣ 온라인 조회

**국민건강보험공단 홈페이지**:
```
1. nhis.or.kr 접속
2. 로그인 (공동인증서/간편인증)
3. 건강검진 > 검진대상조회
```

### 방법 2️⃣ 모바일 앱

**The건강보험 앱**:
```
1. 앱 다운로드 (iOS/Android)
2. 로그인
3. 건강검진 대상 확인
```

### 방법 3️⃣ 전화 문의

```
📞 국민건강보험공단 고객센터
☎️ 1577-1000
```

## 🎗️ 5대 암검진 (무료)

### 암종별 대상자 및 주기

| 암종 | 대상 | 검진 주기 | 검사 방법 |
|------|------|:---------:|-----------|
| **위암** | 만 40세 이상 | 2년 | 위내시경 또는 위장조영 |
| **대장암** | 만 50세 이상 | 1년 | 분변잠혈 (양성 시 대장내시경) |
| **간암** | 고위험군 (B/C형간염, 간경변) | 6개월 | 간초음파 + 혈액검사 |
| **유방암** | 만 40세 이상 여성 | 2년 | 유방촬영술 |
| **자궁경부암** | 만 20세 이상 여성 | 2년 | 자궁경부세포검사 |
| **폐암** | 만 54~74세 고위험 흡연자 | 2년 | 저선량 흉부 CT |

> 🎯 **암검진 대상자**: 출생 연도 + 연령 조건 **둘 다 충족** 필요!

## 🏥 검진 병원 예약 방법

### Step 1️⃣ 검진기관 찾기

```
🔍 국민건강보험공단 홈페이지
→ 건강검진 → 검진기관 찾기
→ 지역/검진 항목별 검색
```

### Step 2️⃣ 예약하기

| 방법 | 설명 |
|------|------|
| **전화 예약** | 병원에 직접 전화 |
| **온라인 예약** | 병원 홈페이지/앱 |
| **방문 예약** | 직접 방문하여 예약 |

### Step 3️⃣ 검진 당일 준비물

```
✅ 신분증 (주민등록증, 운전면허증)
✅ 건강보험증 (없어도 가능)
✅ 검진표 (우편/문자 수신 시)
```
