# 주식 계좌 개설 가이드 2026: 증권사 비교부터 첫 매수까지

주식 계좌 개설 완벽 가이드입니다. 주식 투자, 시작하고 싶지만 **계좌 개설부터 막막**하셨나요? 5분이면 완료되는 비대면 계좌 개설부터 첫 주식 매수까지 알려드립니다.

---

## 3분 요약

- **계좌 개설:** 비대면으로 5분 완료
- **증권사 선택:** 수수료, 앱 UI, 이벤트 확인
- **첫 투자:** 소액(10만원)부터 시작

---

## 주식 계좌란?

### 증권 계좌의 종류

**1. 위탁계좌 (일반 주식 계좌)**
- 국내 주식 매매용
- 가장 기본적인 계좌

**2. 종합계좌**
- 주식 + 펀드 + 채권 통합
- 다양한 금융상품 투자 가능

**3. 연금저축계좌**
- 세액공제 혜택
- 55세 이후 연금 수령

**4. ISA (개인종합자산관리계좌)**
- 비과세/분리과세 혜택
- 다양한 상품 통합 관리

### 초보자 추천 계좌

**위탁계좌로 시작**하세요!
- 가장 기본적인 주식 매매 계좌
- 개설이 가장 간단
- 이후 다른 계좌 추가 개설 가능

### 증권사 선택 기준

**1. 수수료**
- 매매 시 발생하는 비용
- 0.015%~0.1% 범위
- 장기 투자 시 누적 효과 큼

**2. MTS (모바일 앱) 사용성**
- 초보자: 토스증권, 카카오페이 추천
- 중급자: 키움증권 영웅문 추천

**3. 이벤트/혜택**
- 신규 계좌 개설 이벤트
- 수수료 무료 기간
- 주식 지급 이벤트

**4. 리서치/정보 제공**
- 삼성, 미래에셋 리서치 우수
- 투자 정보 필요 시 고려

## 비대면 계좌 개설 방법

### 준비물

**필수**
- 스마트폰
- 신분증 (주민등록증, 운전면허증)
- 본인 명의 은행 계좌 (출금용)

**선택**
- 공동인증서 (없어도 개설 가능)

### 개설 절차 (5분 소요)

**1단계: 앱 설치**
- 원하는 증권사 앱 다운로드
- 예: 토스증권, 키움증권 영웅문S 등

**2단계: 본인 인증**
- 휴대폰 본인 인증
- 신분증 촬영 (앞/뒤면)
- 얼굴 인식 (일부 증권사)

**3단계: 계좌 정보 입력**
- 출금 연결 은행 계좌
- 투자 성향 설문
- 약관 동의

**4단계: 계좌 개설 완료**
- 즉시 사용 가능
- 계좌번호 확인

### 증권사별 개설 특징

**토스증권**
```
토스 앱 → 주식 탭 → 계좌 개설
3분 완료, 가장 간편
```

**키움증권**
```
영웅문S 앱 → 비대면 계좌 개설
5분 완료, 수수료 최저
```

**카카오페이증권**
```
카카오페이 앱 → 주식 탭 → 시작하기
카카오 연동, 소수점 투자 가능
```

## 계좌 개설 후 할 일

### 1. 입금하기

**연결 계좌에서 이체**
1. MTS 앱 실행
2. 입출금 메뉴
3. 연결 계좌에서 이체
4. 금액 입력 후 이체

**추천 첫 입금액**: 10~50만원
- 소액으로 시작
- 연습 후 금액 증가

### 2. MTS 기능 익히기

**필수 메뉴**
- 주식 검색
- 호가창 보기
- 매수/매도 주문
- 잔고 확인
- 체결 내역

**추천 연습**
1. 관심 종목 등록
2. 모의투자로 연습 (일부 증권사)
3. 뉴스/공시 확인

### 3. 첫 주식 매수하기

**종목 검색**
1. 검색창에 종목명 입력
2. 현재가, 등락률 확인
3. 관심종목 등록

**매수 주문**
1. 종목 선택
2. 매수 버튼 클릭
3. 수량 입력 (1주부터 가능)
4. 주문 유형 선택 (시장가/지정가)
5. 주문 실행

## 주문 유형 이해하기

### 시장가 주문

**즉시 체결**을 원할 때
- 현재 시장 가격으로 바로 체결
- 가격 지정 없음
- 빠른 매매 가능

**장점**: 확실한 체결
**단점**: 원하지 않는 가격에 체결될 수 있음

### 지정가 주문

**원하는 가격**으로 주문할 때
- 매수: "이 가격 이하면 사겠다"
- 매도: "이 가격 이상이면 팔겠다"
- 가격 도달 시 체결

**장점**: 원하는 가격에 매매
**단점**: 체결이 안 될 수도 있음

### 초보자 추천

**매수**: 시장가 (확실한 체결)
**매도**: 지정가 (원하는 가격에 매도)

## 세금과 수수료

### 거래 시 비용

**1. 매매 수수료**
- 증권사에 지불
- 0.015%~0.1% (증권사별 상이)
- 매수, 매도 시 각각 발생

**2. 증권거래세**
- 매도 시에만 발생
- 코스피: 0.18% (2026년 기준)
- 코스닥: 0.18%

**3. 양도소득세**
- 대주주만 해당 (소액 투자자 해당 없음)
- 해외 주식: 연간 250만원 초과 수익 시 22%

### 비용 계산 예시

**100만원 매수 → 110만원 매도 시**
```
매수 수수료: 100만원 × 0.015% = 150원
매도 수수료: 110만원 × 0.015% = 165원
증권거래세: 110만원 × 0.18% = 1,980원
---
총 비용: 2,295원
순이익: 10만원 - 2,295원 = 97,705원
```

## 초보자 투자 원칙

### 지켜야 할 5가지

**1. 여유 자금으로만**
- 생활비는 절대 투자 금지
- 잃어도 괜찮은 돈만

**2. 분산 투자**
- 한 종목에 몰빵 금지
- 최소 3~5개 종목에 분산

**3. 분할 매수**
- 한 번에 전액 투자 금지
- 3~5회 나눠서 매수

**4. 장기 투자**
- 단타보다 장기 투자
- 최소 1년 이상 보유 계획

**5. 공부하며 투자**
- 모르는 종목은 투자 금지
- 뉴스, 공시, 재무제표 확인

## 자주 묻는 질문

**Q1. 계좌 개설에 비용이 드나요?**
A. 아니요, 무료입니다. 증권사 계좌 개설은 전부 무료입니다.

**Q2. 여러 증권사 계좌 개설 가능한가요?**
A. 네, 제한 없습니다. 여러 증권사 계좌를 동시에 보유할 수 있습니다.

**Q3. 미성년자도 계좌 개설 가능한가요?**
A. 법정대리인(부모) 동의 시 가능합니다. 비대면보다 영업점 방문이 편리합니다.

**Q4. 해외 주식 투자도 같은 계좌로 가능한가요?**
A. 일부 증권사는 가능합니다. 별도 해외 주식 계좌가 필요한 경우도 있습니다.

**Q5. 계좌 개설 후 바로 투자해야 하나요?**
A. 아니요, 입금하지 않아도 유지 가능합니다. 준비되면 시작하세요.
