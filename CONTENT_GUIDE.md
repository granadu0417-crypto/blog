## 🎯 기존 게시글 개선 과정

### 개선 전 문제점
1. **가독성 부족**: 긴 문단, 띄어쓰기 부족
2. **네비게이션 부재**: 긴 글에서 원하는 섹션 찾기 어려움
3. **내부 링크 부족**: SEO에 불리, 사이트 체류시간 짧음
4. **정보 밀도 낮음**: 빠른 가치 전달 부족

### 개선 작업 (2025-01-05)

#### 1. 목차(TOC) 시스템 추가
**파일**: `components/TableOfContents.tsx` (신규 생성)

**기능**:
- 자동 헤딩 추출 (h2, h3)
- 클릭 시 부드러운 스크롤 이동
- 현재 섹션 하이라이트
- 모바일 반응형

**기술 스택**:
- `remark-rehype`: Markdown → HTML 변환
- `rehype-slug`: 헤딩에 ID 자동 생성
- `rehype-stringify`: HTML 출력 최적화

#### 2. 콘텐츠 구조 개선

**추가된 섹션**:
```markdown
## ⏱️ 3분 요약
- 핵심 내용 3-5줄 요약
- 예상 소요 시간
- 난이도 표시
- 실행 체크리스트

## 📚 목차
(자동 생성 - TableOfContents 컴포넌트)

## 본문 내용
- 단락 구분 명확히 (<br /> 태그 활용)
- 2-3문장마다 줄바꿈
- 이미지 설명 추가
- 체크리스트 활용

## 📚 다음 읽을 글
- 관련 게시글 3개 링크
- 각 글의 요약 (2-3줄)
- 예상 소요 시간, 난이도 표시

## 마치며
- 핵심 내용 재정리
- 실행 단계 요약
- CTA (Call to Action)
```

#### 3. 내부 링크 전략

**규칙**:
- 각 게시글마다 **최소 3개 이상** 내부 링크
- 관련성 높은 게시글 우선 연결
- 자연스러운 문맥에 삽입

**예시**:
```markdown
Next.js로 블로그를 만드는 자세한 방법은
[Next.js 마크다운 블로그 만들기](/posts/nextjs-markdown-blog)를 참고하세요.
```

#### 4. 이미지 처리

**규칙**:
- Unsplash 무료 이미지 활용
- 모든 이미지에 alt 텍스트 (SEO)
- 이미지 설명 추가 (*이미지: ...*)

**형식**:
```markdown
![구글 애드센스 수익화 가이드](https://images.unsplash.com/photo-xxx?w=1200)
*이미지: 블로그 수익화를 상징하는 디지털 마케팅 개념도*
```

## 📊 SEO 최적화 전략

### 키워드 리서치
1. Google Trends 활용
2. 경쟁사 블로그 분석
3. 롱테일 키워드 타겟팅

### 내부 링크 구조
```
메인 페이지 (/)
├── 카테고리 페이지 (/category/tech)
│   ├── 게시글 1 (/posts/nextjs-blog)
│   │   └── 관련 게시글 링크 (3개 이상)
│   └── 게시글 2 (/posts/react-hooks)
│       └── 관련 게시글 링크 (3개 이상)
└── 태그 페이지 (/tag/Next.js)
    └── 해당 태그 게시글 모음
```

### 메타 데이터 최적화
- **Title**: 40-60자, 키워드 포함
- **Description**: 120-150자, 요약 + CTA
- **Keywords**: 5-8개, 관련성 높은 순
